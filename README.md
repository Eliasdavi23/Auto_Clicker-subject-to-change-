# 🖱️ Auto Clicker with GUI

A simple automation tool built with Python that simulates mouse clicks at customizable intervals. The project includes a graphical user interface (GUI) to make configuration easy and user-friendly.

---

## 🚀 Features

- ▶️ Start / Stop auto clicking
- ⏱️ Adjustable click interval
- 🖥️ Graphical user interface (GUI)
- ⚡ Lightweight and fast execution
- 🎯 Simple and intuitive controls

---

## 🧠 Concepts used

- Python automation
- Graphical User Interface (GUI)
- Event-driven programming
- Loops and timing control
- Multithreading (if used to keep UI responsive)
- External libraries integration

---

## 🛠️ Technologies

- Python
- Tkinter (or your GUI framework)
- PyAutoGUI (or equivalent automation library)
